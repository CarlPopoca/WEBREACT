
Abra un cmd y posicionese en la carpeta donde tendra el codigo del aplicativo, y ejecute el siguiente comando:
	
	create-react-app webreactcontactos
	
*******************************************************

Acontinuación algunos comandos para trabajar.

Iniciar el servidor de desarrollo:

	yarn start

Generar el build:

	yarn build

Correr una prueba:

	yarn test

Remover esta herramienta y copias de dependencias de compilación, configuración de archivos y scripts en el 
directorio de la aplicación:

  	yarn eject
********************************************************

Posicionese en la carpeta del código de la aplicación y ejecute los siguientes comandos:

	yarn add react react-dom react-router-dom react-bootstrap bootstrap reactstrap axios react-confirm-alert react-md-spinner

Agregue las siguientes librerias para iconos de forawesome:

	yarn add @fortawesome/fontawesome-svg-core
	yarn add @fortawesome/free-solid-svg-icons
	yarn add @fortawesome/react-fontawesome
	yarn add @fortawesome/free-brands-svg-icons

Importe los estilos de bootstrap en el archivo src/index.js:

	import 'bootstrap/dist/css/bootstrap.min.css';

Importe componentes de reactstrap  en el archivo src/App.js, ejemplo:

	import { Table, Button} from 'reactstrap';

	Nota: Si quiere conocer más acerca de estos comando vaya a la página https://reactstrap.github.io/
